# Photos go here

Drop partner-friendly photos (CrowdRiff-approved, following the brand photography rules) into this folder.

**Naming:** use short, descriptive, lowercase names with hyphens and no spaces, e.g.
`winery-tasting-room.jpg`, `downtown-leesburg-evening.jpg`, `loco-ale-trail-flight.jpg`.

Tell AJ/Claude which photo should go where, or let Claude place them. Likely spots on the site:

- **Hero background** — one strong wide landscape image behind the dashboard headline.
- **"Content we love to reshare" gallery** — a few example photos in the Partner Toolkit showing partners the kind of imagery to send.
- **Section accents** — smaller images alongside the "Why It Matters" content.

Recommended sizes: hero ~2000px wide; gallery/accent ~800–1200px wide. Keep files reasonably compressed (under ~500KB each) so the site loads fast.
