# Logo files go here

Drop the Visit Loudoun logo files into this folder using these **exact file names** so the site picks them up automatically:

| File name | Use | Notes |
|-----------|-----|-------|
| `visit-loudoun-logo.png` | Header logo (light background) | The full-color stacked logo you have works. PNG with transparent background is best. SVG also fine — if you use SVG, name it `visit-loudoun-logo.svg` and tell AJ/Claude to update the one reference. |
| `visit-loudoun-logo-white.png` | Footer logo (dark background) | The **white knockout** version. Optional — if you don't add it, the footer just shows text. |

Once a file with the matching name is in this folder, it appears on the site with no other changes needed. If a file is missing, the site falls back to a styled "VL" text mark, so nothing ever looks broken.
